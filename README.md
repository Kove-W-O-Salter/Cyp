# Cyp

## Version
**0.1.2.4**

## Development Status
**Ready For Use!**

## Installation
To install `Cyp`
on your system run,
with `ruby` installed:
```
> ruby script/install.rb
```
in the project's root
directory.

## Updating
To update your
installation of
`Cyp` run the
following command,
doing what it tells
you to:
```
> ruby script/update.rb
```
